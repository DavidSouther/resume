Karpenter, 2026Now a multi-cloud project (AWS, Azure, GCP, Cluster API…) — but no production-ready path for self-managed k3s on these clouds, so it is cluster-autoscaler per provider. AWS is cleanest; DO needs DOKS. GPU scale-to-zero on DO means destroy/recreate (a powered-off GPU Droplet still bills).
